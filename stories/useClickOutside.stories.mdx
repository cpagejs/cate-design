## useClickOutside 在当前元素之外的区域点击的hook

代码示例
```js
<template>
  <div>
    <div>demo1</div>
    <div ref="demo2Ref">demo2</div>
  </div>
</template>

<script setup>
  import { ref } from "vue";
  import { useClickOutside } from "cate-design";

  const demo2Ref = ref();

  useClickOutside(demo2Ref, () => {
    console.log("useClickOutside")
  });
</script>
```

参数说明

| 参数 | 说明 | 
| - | - | 
| ref | ref对象 | 
| callback | 回调函数 |
