## useDebounce 防抖hook

代码示例
```js
<template>
  <div>
    <cInput v-model="inputValue"/>
  </div>
</template>

<script setup>
  import { ref } from "vue";
  import { useClickOutside, cInput } from "cate-design";

  const inputValue = ref("123");

  const newVal = useDebounce(inputValue.value, 100);
</script>
```

参数说明

| 参数 | 说明 | 
| - | - | 
| value | 监听的值 | 
| timer | 延迟时间，默认200毫秒，单位毫秒 |
